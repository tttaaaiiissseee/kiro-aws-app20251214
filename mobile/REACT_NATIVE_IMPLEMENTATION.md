## 🔧 開発環境セットアップ

### 必要な環境
```bash
# Android開発
- Android Studio
- Java 17+
- Android SDK

# iOS開発 (macOSのみ)
- Xcode
- CocoaPods
```

### 実行方法
```bash
# 依存関係インストール
cd mobile
npm install

# iOS実行 (macOSのみ)
npm run ios

# Android実行
npm run android

# Metro bundler起動
npm start
```
